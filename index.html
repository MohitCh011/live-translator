Welcome to AetherTranslate

An advanced client-side translation assistant for Telugu speech. Speak in Telugu, and see English subtitles formed in real-time inside the floating glass window.

1

Configure Keys

Open Settings to set up Gemini or OpenAI API keys for perfect sentence translation. For quick tests, the Free mode works out-of-the-box.

2

Position Window

Drag the Floating Subtitle Window by its header to position it anywhere. Resize it by dragging the window boundaries or corners.

3

Start Speaking

Click "Start Subtitles" above and speak. English subtitles will stream live, showing up to 5 lines with autoscrolling.

Test Offline Speech File

Select your recorded offline speech file (such as your WhatsApp audio) to play and test the live subtitles routing.
